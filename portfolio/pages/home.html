Unterrogation

...

A top-down 2D narrative game developed in Unity where you interrogate crew members on an intergalactic investigation space station. Players select from dialogue options to learn more information and utilize special technology to wipe the memories of crew members in order to ask different questions and catch them in their lies.

  • Headed the technical design of the game code.
  • Integrated Ink by Inkle, a tool that our writer could use to quickly integrate branching dialogue scripts.
  • Programmed the feature to confront NPCs when they are caught lying.
  • Created an interactive dialogue system.
  • Crafted systems enabling players to reset their conversations with NPCs using the memory-wipe.
  • Implemented a game saving system.
  • Built software architecture to manage game state.
  • Play the game!
